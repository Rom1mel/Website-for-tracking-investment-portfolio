## Цель проекта:
Разработка и внедрение рабочего MVP веб-сервиса C&D Invest, который предоставит пользователям возможность централизованно отслеживать и анализировать гибридные инвестиционные портфели, объединяющие традиционные (акции, ETF) и цифровые (криптовалюты) активы, через интуитивно понятный веб-интерфейс.
